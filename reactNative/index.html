App Mobile de Pedidos

Desarrollo de aplicación movil, para sistemas Android, utilizando la tecnología de React Native.

Tareas: Maquetado, estilos y diseños, Integración de paquetes para Componentes, Captura y Validación de Datos, Manejo de respuesta del Servidor (archivos.json).
Tecnologías: React Native, Css, React Native Paper, Gestor de paquetes NPM, Entorno Expo Go, Git y GitHub.
