Widget del Clima

0

El proyecto consiste en mostrar en un estilo de widget el clima, la hora y fecha actual.

Para el desarrollo se utilizan tecnologías como:

HTML y CSS para la estructura y diseño.

JAVASCRIPT para obtener la hora y fecha, así también para el consumo del WeatherAPI.com y obtener el clima actual.

Manipulación del DOM para editar en tiempo real los estilos.
